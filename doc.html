在确认本协议之前，请您仔细阅读并充分理解本协议，特别是以粗体及/或下划线标注的内容,以及免除或限制满帮的责任、限制您的权利、规定争议解决方式的相关条款。如果您不同意本协议的任意内容，或者无法准确理解本协议任何条款的含义，请不要进行确认及后续操作。

1.1  满帮平台/平台：指由江苏满运软件科技有限公司（“运满满”）所运营的“运满满司机”、“运满满货主”APP/PC端等相关网络平台，以及由贵阳货车帮科技有限公司运营“货车帮司机”APP、“货车帮货主”APP/PC端以及货车帮企业版等软件/网站。您可通过满帮平台向贷款方提出贷款审批、查看额度、申请借款等操作，具体以实际用于向您提供本协议项下的某项特定服务的平台为准。

1.2  用户：指符合相关条件、注册满帮平台并使用平台贷款综合服务的法人、自然人或其他组织。

1.3  征信记录：包括但不限于您的经营状况、信用状况、涉诉情况、您的姓名、收入、存款、有价证券、商业保险、不动产的信息和纳税数额信息等财产状况、身体健康状况、位置、行为记录、失信违约记录、违法违规记录等敏感信息等。

1.4  平台贷款综合服务：指满帮通过满帮平台为您向适格的贷款主体申请贷款提供平台技术和信息传递服务，并由满帮旗下符合其资质的满帮小贷公司及/或其合作的符合资格的贷款方（以下简称“贷款方”）、担保机构（以下简称“担保方”）提供贷款相关服务。贷款方在对用户信息进行大数据分析基础上，给用户一定的授信额度（以下简称“授信额度”），在满足一定条件的前提下，您在该授信额度、授信期限内，可获得消费信用借款（以下简称 “消费借款”或“借款”），具体以满帮实际为您推荐并展示的贷款产品信息为准。

1.5  融资专用账户：指用于接收用户通过平台贷款综合服务获得的借款的收款账户，具体账户以实际用作接收用户借款的账户为准。其中，若您的消费服务提供方（包括但不限于ETC卡的发行方及/或其代理机构、油站等）已指定或明确专门的账户用作接受您的消费款项的，则该等账户即为融资专用账户，以便确保用户获得的借款专款专用。

1.6  融资协议：指用户与贷款方、担保方（如有）签订的借款相关协议，具体协议名称以用户和贷款方实际签订的为准。

1.7  还款日/账期截止日：根据用户与贷款方签订的融资协议的约定，用户应当向贷款方偿还借款的日期/正常还款的截止日。

1.8  满帮：在本协议中指满帮平台的运营主体及其关联公司，以及指满帮旗下合法设立运营并取得合法资质的小额贷款公司、合法设立运营并去的合法资质的担保公司（如涉及），包括但不限于贵州货车帮小额贷款有限公司、贵州帮货车融资担保有限公司、贵阳货车帮科技有限公司、成都运力科技有限公司、上海细微信息咨询有限公司和江苏满运软件科技有限公司等。贵州货车帮小额贷款有限公司将代表满帮与您签署本协议。

 

2.  申请开通平台贷款综合服务的条件

2.1  您确认已根据满帮平台发布的相关规则和相应流程，注册成为满帮平台的用户。

2.2  您在开通、申请使用贷款综合服务的过程中，需按照满帮平台提示的内容提供个人身份信息等相关资料（以下统称“个人资料”），以完成激活过程。

2.3 您需保证在激活过程中填写的各项内容和提交的资料的正确性、真实性、准确性、完整性和有效性。

2.4  您个人资料如有变动的，应及时根据满帮平台所提供的更新方式进行信息的更新。您信用情况、财务状况或还款能力如有变动的，应及时告知满帮。若您提供任何错误、不真实、不准确、不完整、过时的信息或在信息发生变动后未能及时更新并说明，或者满帮有合理理由怀疑您提供的信息为不正确、不真实、不准确、不完整或并非最新有效，满帮有权停止向您提供平台贷款综合服务。您同意，满帮就前述停止服务的行为不承担任何责任，您将承担因此产生的任何直接或间接支出、损失及其他不利影响或后果包括但不限于对您使用平台贷款综合服务及/或借款产品造成影响。

2.5  您务必妥善保管您的满帮平台的账户及密码，应在知晓账号被盗后第一时间通知满帮平台。

2.6  在满足本协议及满帮平台发布的其他规则的前提下，确认签署本协议，即可申请使用平台贷款综合服务，并通过平台贷款综合服务向贷款方申请借款。

 

3.  平台贷款综合服务的使用及管理

3.1  您通过平台贷款综合服务申请借款的，贷款方、担保方（如有）将根据您的授权查询您的征信记录，并结合您的满帮平台业务使用情况，通过大数据分析，对您的信用情况进行评估，从而通过满帮平台通知您是否取得授信，并在给予您授信的情况下，确定您可使用授信的用途。您与满帮签署本协议并不代表您一定能成功取得贷款方的授信。

3.2  贷款方同意授信的，会给您一个授信额度和授信期限，并即时显示在您的平台借款服务业务界面。您获得授信额度后，在该等额度内，您有权通过满帮平台向贷款方发起借款申请。根据需要，贷款方有权限制您使用授信的用途。但，贷款方是否提供借款、借款用途以及其他借款相关条件由贷款方审核后决定，您与满帮签署本协议并不代表您一定能获得借款，或获得的借款金额、借款用途与您申请的一致。

3.3  在您使用平台贷款综合服务及/或申请借款的过程中，根据您的资信状况、借款使用情况和风险信息等情况，贷款方可能会随时对您的授信额度、授信期限等事项进行调整或随时撤回授信并拒绝向您发放借款,具体调整规则由您与贷款方另行签订协议确定，满帮不对您的授信额度和授信期限的获取和调整做出任何担保和承诺，不承担任何义务。

3.4  您同意以融资专用账户接收您通过平台贷款综合服务申请到的借款，用于您的ETC卡充值、油费支付、保费支付等（具体以您选择并经贷款方确认的借款用途为准），确保专款专用。贷款方将借款支付至融资专用账户，即视为您已收到借款。

3.5  如满帮与您就特定产品或服务相关的平台贷款综合服务另行签署具体的协议（“具体协议”）且该具体协议中约定的包括成功放款的标准等内容与本协议不一致的，以具体协议为准。

3.6  您应按照满帮平台提示绑定银行卡作为还款账户，并在此不可撤销地同意满帮或合作的贷款方、担保方（如有）通过合作的第三方支付机构或银行（合称“支付机构”）在还款日/账期截止日前即可直接从该等还款账户中代扣您通过平台贷款综合服务申请的借款本金、利息、罚息、违约金与催收费用（如有）、手续费（如有）、担保费（如有）等应还款项（合称“应还款项”），实现还款，恢复授信额度。您应确保上述还款账户内的资金余额自还款日/账期截止日前一个自然日起即应不少于应还款项。

3.7  如您不再使用平台借款综合服务而注销的，注销前已发生的债权债务关系不受注销的影响。在前述情况下，贷款方有权通过满帮通知您注销之前产生的所有应还款项在注销当时提前到期，并要求您在收到通知之日内清偿完毕该等所有欠款，或应通过其它途径及时还款。您知晓并同意上述代扣服务在完成结清前产生的逾期利息、罚息、违约金、催收费用等逾期息费由您承担，上述代扣服务将在结清前持续进行。

 

4.  还款

4.1  您还款的一般清偿顺序为：担保费（含违约金，如有）/服务费（如有）、应向贷款方清偿的应还款项，具体以您签署的融资相关协议为准。

4.2  为方便您的使用，满帮平台在进行页面展示时，可能将利息、担保费（如有）及其它费用（如有）进行整体展示，如您需要各部分具体明细，可通过在线客服了解。

4.3  在还款日/账期截止日前，您绑定的用于偿还借款的银行卡金额不能足额清偿当期应还款项（包括但不限于应向贷款方清偿的款项、应向担保方支付的款项（如有）等）的，满帮无法完成还款代扣且您未通过其它满帮平台公式的方式足额清偿的，视为您逾期还款。您连续两次或两次以上逾期的或逾期清偿任一期应还款项超过___天的，满帮有权停止您使用平台贷款综合服务及/或申请借款，要求您赔偿满帮由此受到的全部损失，并且，贷款方、担保方（如有）或满帮亦有权根据融资协议的约定要求您支付逾期罚息/催收费用、借款利息、借款本金、违约金、担保费等并承担其他违约责任；为避免歧义，如满帮、担保方（如有）自贷款方处承继或获得了应还款项债权，则该等权利由满帮、担保方（如有）享有。
